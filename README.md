# DigiTools - Premium Digital Tools Store

Digitools is a modern and responsive e-commerce storefront dedicated to digital software and tools. Built with React.js, it offers a seamless browsing experience where users can explore premium subscriptions and digital assets with a clean, intuitive UI.


## 🛠️ Tech Stack 

* JavaScript (Es6+)
* React.js
* Tailwind Css
* daisy UI
* React Toastify (NPM Package)


## ✨ Features

* Responsive & Modern UI: Optimized for all devices using a clean and intuitive design.
* Cart Management: Seamlessly add to cart and remove cart feature.
* Interactive Notifications: Integrated React Toastify to provide real-time feedback for user actions.
